# todopoint
tg: @todopoint_bot

Простой туду бот с начислением очков за закрытые таски

### **ver 1.0**

4 кнопки из начального меню
1. новый таск
    можно записать название нового таска (отработка пустых строк); + 20 очков за каждый новый таск 
2. закрыть таск
    меню с выбором таска для закрытия (+100 очков за каждый)
3. таски \ статистика
    количество очков, тасков; список тасков
4. настройки
    показывает очки и время уведомлений, есть возможность сбросить очки до 0 и отключить уведомления

Потенциально хочу добавить кастомное время уведомлений, глубже продумать систему очков, сделать нормальный пункт статистики
